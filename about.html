Project Information

SocioX is a college project developed to demonstrate modern web development practices and social media platform architecture. This project showcases the implementation of a complete social networking website using fundamental web technologies.

Technologies Used

HTML5 CSS3 Responsive Design Web Standards

Key Features

SocioX offers a comprehensive set of features designed to enhance your social networking experience:

  • User profiles with customizable information
  • News feed with real-time updates
  • Post creation and sharing capabilities
  • Interactive engagement through likes and comments
  • Clean and minimalist user interface
  • Responsive design for all devices
